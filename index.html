StrataVariance Labs (SVL)

Keel Framework — Reciprocal Validation Within Bounds

A diagnostic layer for measuring reasoning drift in generative systems.

What Keel does

  • Forward → Reciprocal → Alignment → RDL (Reciprocal Deviation Log)
  • Metrics: SIC (semantic overlap), Δ (drift), τ (order distance)
  • Diagnose, don’t decide: no in-line correction

Whitepaper

Open Definition (KOD-1.0)

Use, adapt, and extend Keel with attribution. No warranty, no certification. Read KOD-1.0

Quickstart

pip install keel-svl
keel run prompt.txt --bounds 0.8 --out "runs/$(date -u +%FT%TZ)/rdl.json"

Artifacts: forward.txt, reciprocal.txt, align.json, rdl.json, telemetry.json.
