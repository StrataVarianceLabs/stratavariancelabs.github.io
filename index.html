Keel v3.3.2 — Five Goldens (Single-Cycle Demo)

Keel is a verification framework for generative and agentic systems. It runs paired, bounded reasoning passes (forward and reciprocal) and measures variance rather than correcting output. Keel emits a manifest, a reciprocal deviation log (RDL), and telemetry so teams can assess consistency within declared bounds — not external truth.

This release demonstrates:

  • Five golden consistency cases (G01–G05)
  • Forward + reciprocal cycle for each case
  • Shim hallucination / drift summary for reproducibility
